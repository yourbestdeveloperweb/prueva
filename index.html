ACTIVIDAD 1 CSS (Valor 10 puntos).

1-¿Qué es CSS?

CSS (Cascading Style Sheets) es un lenguaje de estilo que se utiliza para describir la presentación de documentos HTML

2-¿Cuál es la función principal de CSS?

La función principal de CSS es separar el contenido del diseño, permitiendo controlar la apariencia de elementos como colores, fuentes y layout.

3-¿Cómo se puede incluir CSS en un documento HTML?

Se puede incluir CSS de tres maneras: en línea (atributo style), en la sección < style> dentro del < head>, o mediante un archivo externo con < link>.< /p>

4- ¿Qué son los selectores en CSS?

Los selectores son patrones utilizados para seleccionar los elementos HTML a los que se aplicarán las reglas de estilo.

5-¿Qué son las propiedades y valores en CSS?

Las propiedades son características que se pueden modificar (como color, font-size), y los valores son las especificaciones para esas propiedades (como red, 16px).

6-¿Cómo se aplica el modelo de caja en CSS?

El modelo de caja describe cómo se representan los elementos en la página: cada elemento tiene márgenes, bordes, rellenos y el contenido en sí.

7-¿Qué es el "cascading" en CSS?

"Cascading" se refiere a cómo se aplican las reglas de estilo en orden de prioridad, donde las reglas más específicas o recientes sobreescriben las más generales.

8-¿Qué son las media queries?

Las media queries son reglas que permiten aplicar diferentes estilos dependiendo de las características del dispositivo, como el tamaño de la pantalla.

9-¿Cuáles son algunos estilos comunes que se pueden aplicar?

Estilos comunes incluyen color de fondo, tipos de letra, márgenes, padding, alineación, bordes y efectos de transición.

10-¿Cómo se puede usar CSS para crear un diseño responsivo?

Utilizando media queries para ajustar el diseño según el tamaño de la pantalla y unidades relativas (como % y vw) para elementos fluidos.
